
=================
SQL et Map Reduce
=================

Le langage :epkg:`SQL` est très proche du paradigme
:epkg:`map/reduce`. Le langage :epkg:`Python` inclut
le module :epkg:`sqlite3` qui permet de manipuler des
bases de données en local. Cela se révèle assez pratique
pour dépasser la capacité de la mémoire d'un ordinateur
locale. Il est également possible de créer ses propres
*mappers* et *reducers* avec cet outil.
